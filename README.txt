Personal Finance Tracker 
------------------------

Personal Finance Tracker is a simple terminal based dashboard.
The main purpose is to track your transactions. 
You can search/filter and even make reports. (Features to be added)



Usage
-----

To use the program, open and run in a linux based terminal (e.g Ubuntu)
You will be prompeted with a menu and to enter your choice.

# This allows you to exit/terminate the program
0. Exit  

# This opens up a secondary menu to manage transactions (add, delete, edit and view)
1. Manage Transactions

# This opens up a secondary menu to search for specific/filter transactions. filter by Date/Category/Amount/Type
# Feature yet to be added
2. Search/Filter Transactions

# This opens up a secondary menu to view different types of reports and Summaries
# Feature yet to be added
3. Reports & Summaries
